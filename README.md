Cleaning Data Course Project
============================

*run_analysis.R*

Only one script is included which performs all the tasks.

**Requirements**

The data to be processed must be in the directory: *data-raw/*. If it is in a different place, the path can be defined by changing the variable *dataDir* in the script.

**Running**
```
> source("run_analysis.R")
```
